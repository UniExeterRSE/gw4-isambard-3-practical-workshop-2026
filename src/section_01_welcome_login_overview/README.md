# Section 1: Welcome, Login, and System Overview

This section covers the opening 30 minutes of the workshop: a 10-minute BriCS introduction (delivered by BriCS staff),
followed by login and a lightweight system overview while attendees get connected.

Files in this section:

- `00-pre-session.markdown` — pre-session arrival slide (09:30–10:00 display), portal slide, login-start choreography,
  VS Code signpost, and follow-along contingency. Contains `TODO` placeholders for exact presenter wording and confirmed
  commands.
- `01-system-overview.markdown` — main system overview slide deck: what Isambard 3 is, storage areas, workflow loop,
  fit-check, and getting-help slide.
- `02-grace-cpu-superchip.markdown` — deep-dive slide deck on the NVIDIA Grace CPU Superchip (architecture, NUMA, memory
  subsystem, vectorization, peak FLOPS).
- `02-grace-cpu-superchip.md` — attendee-facing prose reference covering the same Grace CPU content.

Adaptation notes:

- This workshop is for **Isambard 3**, not Isambard AI.
- Isambard 3 is **CPU-only** and uses **Arm/aarch64**.
- Keep the system overview light; defer hands-on commands to Section 2.
- The 10-minute BriCS introduction at the start of this section is delivered by BriCS staff; its content is not managed
  here.
