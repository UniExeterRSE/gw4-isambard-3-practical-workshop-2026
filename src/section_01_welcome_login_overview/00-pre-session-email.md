**To:** Registered attendees\
**Subject:** Isambard 3 Practical Workshop — setup steps to complete before Tuesday

------------------------------------------------------------------------------------------------------------------------

Dear all,

Thank you for registering for the **Isambard 3 Practical Workshop** on **Tuesday 21 April, 10:00am–12:30pm** at the AI
Arena, SWIoT, Exeter.

To make the most of our time together, please complete the following three setup steps before you arrive. Full
instructions are on our pre-session page:

<https://uniexeterrse.github.io/gw4-isambard-3-practical-workshop/section_01_welcome_login_overview/00-pre-session.html>

1.  **Initial account setup** — create your account on the Isambard 3 portal and accept the workshop project invitation
    (watch your email — the invitation will arrive shortly).
2.  **Clifton and SSH** — install the Clifton authentication tool on your laptop and verify you can log in to Isambard
    3.  
3.  **VS Code CLI** — install the VS Code CLI on Isambard 3 so you have a working editor ready for the session.

The pre-session page above walks through each step. If you are an advanced user and prefer your own editor or
environment, that is fine — bring it set up and ready to use.

**Please bring a laptop** with a terminal and a web browser.

**Doors open at 09:30** for tea and coffee. If you have not yet completed the setup steps, arrive from 09:30 and work
through them with a helper in the room — it is exactly the same process, just with in-person support. If everything is
already working, there is no need to arrive before 10:00. **Workshop teaching starts at 10:00.**

If the project invitation has not arrived, or anything else has gone wrong, **please reply to this email before the
day** so we can investigate in advance.

We look forward to seeing you on Tuesday.
