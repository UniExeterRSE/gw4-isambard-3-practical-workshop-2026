# Section 4: Installing Software

This section is the main home for the original tutorial’s software-installation material, adapted for Isambard 3 and for
this workshop schedule.

Files in this section:

- `01-modules-first.md` - decision order and modules workflow
- `02-conda-environment.md` - user-managed Python environment exercise
- `environment.yml` - example conda environment file
- `check_scipy.py` - minimal Python check script
- `run_python_env_check.sh` - batch job for the Python example
- `03-containers-follow-up.md` - optional follow-up reading, not part of the core taught path

Adaptation notes:

- All examples are for **CPU-only Isambard 3**
- GPU-specific batch directives from the original tutorial have been removed
- Containers are included only as a brief follow-up route, matching the workshop plan
