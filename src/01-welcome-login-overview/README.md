# Section 1: Welcome, Login, and System Overview

This section adapts the introductory parts of the BriCS "Installing and Running Software" tutorial for the Isambard 3 workshop.

Use these files during the opening 30 minutes while attendees are logging in and getting oriented.

Files in this section:

- `01-system-overview.md` - presenter-facing overview notes and slide content
- `02-storage-worksheet.md` - a short attendee worksheet on storage spaces

Adaptation notes:

- This workshop version is for **Isambard 3**, not Isambard AI.
- The important platform detail is that Isambard 3 is **CPU-only**.
- Isambard 3 uses **Arm/aarch64**, which matches the architecture used elsewhere in BriCS.
- We keep this section light and defer hands-on commands to Section 2.

Original source:

- https://docs.isambard.ac.uk/user-documentation/tutorials/intro-tour/
